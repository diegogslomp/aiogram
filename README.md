### Finite State Machine Telegram Bot

Standalone
```
git clone --single-branch https:/github.com/diegogslomp/aiogram bot
cd bot
pip install aiogram
# Add _env keys to environment
python bot.py
```

Django
```
cd django-project
git clone --single-branch https:/github.com/diegogslomp/aiogram bot
pip install aiogram
# Add _env keys to environment
# Add bot app inside project config
python run manage.py bot
```
